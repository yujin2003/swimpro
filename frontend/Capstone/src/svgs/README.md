# SVG 및 이미지 파일 다운로드 안내

이 폴더에 Google Drive에서 다음 파일들을 다운로드해서 넣어주세요:

## 필수 파일 목록

### 기본 베이스 이미지 (PNG)
- `baseA.png` - 전면 기본 인체 모형
- `baseB.png` - 후면 기본 인체 모형

### 영법별 근육 SVG
- `freeA.svg` - 자유형 전면 근육
- `freeB.svg` - 자유형 후면 근육
- `backA.svg` - 배영 전면 근육
- `backB.svg` - 배영 후면 근육
- `breaststrokeA.svg` - 평영 전면 근육
- `breaststrokeB.svg` - 평영 후면 근육
- `butterflyA.svg` - 접영 전면 근육
- `butterflyB.svg` - 접영 후면 근육

## 다운로드 방법

1. Google Drive 링크 열기: https://drive.google.com/drive/folders/1ZKfQ7o3nFqXZ_cIQE5-MwZM5A2hvW-dU?usp=sharing
2. 각 파일을 개별적으로 다운로드하거나, 전체 폴더를 다운로드
3. 다운로드한 파일들을 `src/svgs/` 폴더에 복사

## 파일 위치 확인

모든 파일이 `src/svgs/` 폴더에 올바르게 위치했는지 확인하세요.
